Projects

Blackjack on Kubernetes

A playable blackjack game written in Python (Flask), containerized with Docker and deployed on a self-managed Kubernetes cluster using MicroK8s — the whole deployment defined as YAML manifests. An nginx ingress with sticky sessions keeps each player pinned to one pod, while a HorizontalPodAutoscaler scales the dealer from 1 to 10 replicas as CPU load climbs, with liveness and readiness probes on a /health endpoint.

Mini ERP System

A full-stack inventory management system built as a school project: a Node/Express REST API secured with JWT auth (bcrypt-hashed passwords), a React front end, and a SQLite database. It covers user accounts, item management, live stock adjustments (restock and consume), search, and CSV export — full CRUD from end to end.
